# MediCI - Sistema Inteligente de Citas Médicas

## Descripción del proyecto

MediCI es un sistema web inteligente para el agendamiento automático de citas médicas en un entorno hospitalario. El objetivo del proyecto es reducir el tiempo promedio de registro de citas y disminuir errores de empalme de horarios mediante un módulo de asignación automática.

## Problema

En muchos hospitales o clínicas, el registro de citas médicas se realiza de forma manual mediante llamadas, hojas de cálculo o libretas. Este proceso puede generar pérdida de tiempo, errores de captura, citas duplicadas y conflictos de horario entre pacientes, médicos y consultorios.

## Hipótesis

La implementación de un sistema automático de asignación de citas médicas reducirá en un 40% el tiempo promedio de registro de una cita y disminuirá en un 50% los errores de empalme de horarios en comparación con el método manual tradicional.

## Objetivo general

Desarrollar un sistema web inteligente de agendamiento automático de citas médicas para optimizar el registro de consultas, reducir tiempos administrativos y disminuir errores de empalme de horarios en un entorno hospitalario.

## Tecnologías propuestas

* React
* Node.js
* Express
* MySQL
* GitHub
* Excel o Python para análisis de resultados

## Módulos principales

* Login de usuario
* Registro de pacientes
* Registro de médicos
* Registro de especialidades
* Registro de consultorios
* Gestión de citas médicas
* Asignación automática de horarios
* Validación de empalmes
* Registro de logs
* Dashboard de métricas

## Métricas a medir

* Tiempo de registro de una cita
* Errores de empalme
* Citas registradas correctamente
* Intentos fallidos
* Horarios sugeridos por el sistema
* Citas canceladas o reprogramadas
